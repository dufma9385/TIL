#### 사용자 관리

- adduser username = 사용자 생성
- passwd = 사용자 비밀번호 변경
- usermod 사용자의 속성 변경
- userdel 사용자 삭제
- chage 사용자의 암호를 주기적으로 변경하도록 설정

#### group 관리

- groups 사용자가 소속된 그룹표시
- groupadd 그룹생성
- groupmod 그룹의 속성변경 
- groupdel 그룹삭제
- gpasswd 그룹의 암호설정 , 그룹관리 수행



각 파일마다 사용자가 읽을 수 있는 권한이 있다 

**chmod 777**의 경우 모든 사용자가 파일을 읽고 수정할 수 있다
